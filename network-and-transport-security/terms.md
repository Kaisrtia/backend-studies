##### Core Protocols
* **TLS / SSL**: Transport Layer Security / Secure Sockets Layer
* **HTTPS**: HTTP Secure (Uses port 443)
* **SSH**: Secure Shell (Secure remote access, uses port 22)
* **IPsec**: Internet Protocol Security (IP layer security)
* **WPA3**: Wi-Fi Protected Access 3 (Next-generation wireless security)

##### Concepts & Techniques
* **Data in Transit**: Data actively moving across a network
* **Tunneling**: Creating a secure, encrypted communication tunnel
* **VPN**: Virtual Private Network (Often uses IPsec)
* **Handshake**: The process of establishing a connection (e.g., TLS Handshake, TCP 3-way Handshake)
* **Certificate**: Digital security certificate used for authentication (e.g., X.509)

##### Attacks Prevented
* **Sniffing / Eavesdropping**: Secretly intercepting network traffic
* **MITM (Man-in-the-Middle)**: Intercepting and possibly altering communication between two parties
* **Brute-force**: Password guessing attacks (Effectively mitigated by WPA3 on Wi-Fi networks)
